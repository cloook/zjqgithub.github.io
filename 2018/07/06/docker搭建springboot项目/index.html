docker搭建springboot项目

2018-07-06

参考自springboot官方demo

创建应用

创建一个基本的springboot项目，修改pom文件.

这里要注意Dockerfile要放在最外层

编辑 Dockerfile

把项目打包

mvn package

使用 DockerFile 构建镜像

mvn package docker:build

构建好之后在docker中查看

docker images

启动镜像

docker run -p 8080:8080 -t springio/demo
