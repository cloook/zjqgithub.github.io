优雅地遍历数组

2017-07-03

原文地址:https://hyjk2000.github.io/2011/06/09/optimizing-javascript-array-traversing/

##优化一
先把数组的长度先查出来，存进一个局部变量，那么循环的速度会大大提高

##优化二
我们还可以让它更快。如果循环条件不需要进行比较运算，那么循环的速度还可以更快

#优化三
将优化一与优化二结合起来，用一种优雅而高效地方式去遍历吧！
